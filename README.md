# insta-scraper

A Instagram scraper written in Python that scrap (insta ID, Followers, Following, etc) 

# How to install

## In Termux
`pkg install requests` 

`pkg install bs4` 

## In Linux and windows
`pip install requests` 

`pip install bs4`
